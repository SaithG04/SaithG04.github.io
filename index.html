VICTORIUS

Este proyecto es una aplicación para la gestión del minimarket-licorería "Victorius", desarrollada en Java. Está diseñada para manejar operaciones esenciales como la gestión de productos y pedidos, con una integración robusta para acceder y manipular datos en una base de datos MySQL utilizando el patrón de diseño DAO. El sistema permite añadir, modificar, eliminar y visualizar productos, así como registrar y administrar pedidos, asegurando un flujo eficiente de información. La estructura del proyecto es modular, con una capa de acceso a datos que gestiona la conexión y las operaciones CRUD sobre las entidades principales. Además, el uso de Apache Maven para la gestión de dependencias y la configuración del proyecto garantiza una construcción y mantenimiento simplificados, haciendo de este sistema una solución práctica y eficiente para la administración de "Victorius".

Momentos

"Momentos" es una aplicación web desarrollada en PHP para la empresa "Momentos E.I.R.L", que facilita la gestión y creación de pedidos personalizados para eventos especiales como bodas, quinceañeras, graduaciones, y otros momentos importantes. La aplicación permite a los usuarios iniciar sesión, gestionar sus pedidos y generar documentos PDF con los detalles del evento. El proyecto está estructurado para manejar diferentes tipos de pedidos a través de controladores específicos, asegurando una experiencia de usuario fluida y personalizada. Con una interfaz intuitiva y funcionalidades clave como la descarga de PDFs, "Momentos" es una herramienta esencial para planificadores de eventos y usuarios que buscan organizar de manera eficiente sus celebraciones importantes.

EcoRuta

"EcoRuta" es una innovadora aplicación móvil diseñada para mejorar la interacción entre los ciudadanos y el servicio de recolección de basura en su ciudad. La aplicación permite a los usuarios ver en tiempo real la ubicación del camión de basura y recibir notificaciones cuando este se encuentre cerca de su domicilio, facilitando así la gestión de sus desechos. Además, los usuarios pueden escanear códigos QR para acumular puntos diarios que pueden ser canjeados por premios en el futuro. La app también ofrece la opción de registrarse mediante un correo verificado o a través de Google, y permite reportar problemas de contaminación subiendo fotos directamente desde la aplicación. "EcoRuta" es una herramienta esencial para fomentar la participación ciudadana en la gestión de residuos y contribuir a una ciudad más limpia y sostenible.

PaddyRice Production Manager

"PaddyRice Production Manager" es una aplicación diseñada para optimizar y gestionar el proceso de producción de arroz. Desarrollada en Java, esta herramienta proporciona a los productores de arroz la capacidad de calcular la demanda, generar reportes en formato PDF, y administrar diferentes aspectos del proceso de producción a través de una interfaz intuitiva. El sistema está organizado en diferentes capas que incluyen la lógica de negocio, acceso a datos, y utilidades comunes, permitiendo a los usuarios gestionar de manera eficiente las operaciones diarias en la producción de arroz.

Vitality

Es una aplicación web desarrollada para el Centro Médico "Vitality" con el propósito de gestionar las capacitaciones del personal médico. La plataforma permite a los administradores del centro organizar y administrar cursos y especialidades dirigidos al personal, facilitando la actualización continua de sus conocimientos y habilidades. Además, ofrece módulos específicos que permiten la gestión eficiente de la información del personal y de los programas de formación, con interfaces personalizadas para diferentes roles dentro del centro, como administradores, docentes y alumnos. "Vitality" es una herramienta esencial para asegurar que el personal médico del Centro Médico "Vitality" esté bien capacitado y preparado para ofrecer la mejor atención posible.

RENEWAL'S END

"Renewal's End" es un juego de aventuras y puzzles desarrollado en Unity y programado en C#, que aborda una temática ambiental crítica. En un futuro devastado por la contaminación, la Tierra se ha vuelto inhabitable, y la única esperanza de la humanidad reside en una nave espacial construida por el padre del protagonista, ahora desaparecido. Los jugadores deberán asumir el papel del protagonista, quien debe reparar y hacer funcionar la nave para alcanzar un planeta habitable. A lo largo del juego, los jugadores resolverán una serie de puzzles que desbloquean rutas hacia otros planetas, cada uno presentando nuevos desafíos y misterios. "Renewal's End" no solo ofrece una experiencia de juego inmersiva, sino que también invita a reflexionar sobre el impacto del cambio climático y la importancia de cuidar nuestro planeta. El juego está diseñado para PC y combina gráficos creados con Cinema 4D con una narrativa profunda, ofreciendo una experiencia única y significativa para los jugadores.
